# 🌌 Bilgihavuzu - Cyberpunk Online Exam & Quiz Platform

PHP 8.x ve MySQL PDO altyapısıyla geliştirilmiş, ön yüzü **Dark Neon / Glassmorphism** tasarım trendine uygun olarak modernize edilmiş dinamik bir çevrimiçi sınav ve deneme platformu.

## 🚀 Özellikler
- **Gelişmiş Öğrenci Portalı:** Canlı istatistikler, geçmiş sınav analizleri ve PDF çıktı motoru.
- **Dinamik Sınav Konfigüratörü:** JavaScript Fetch API (AJAX) ile sayfa yenilenmeden dinamik konu ve soru seçimi.
- **Siber Kontrol Merkezi (Admin Panel):** Chart.js entegreli neon grafikler, CSV ile toplu soru içe aktarma altyapısı.
- **Üst Düzey Güvenlik:** %100 PDO Prepared Statements ile SQL Injection koruması ve Bcrypt şifre hashleme.
